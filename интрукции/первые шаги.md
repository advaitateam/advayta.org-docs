- Создайте учетную запись в [github](https://github.com/) и сообщите ваш никнайм или емайл Видьяранье
- Откройте Obsidian. Создайте vault c названием `advayta-org`, откройте настройки, включите галачку community plugin ![image](https://media-thumbs.advayta.org/m/1286a970c5145f73a9cd9bdca886219d.png)
- Нажмите browser, найдите установите плагин git
  ![image](https://media-thumbs.advayta.org/m/ffd4ef8d23d0a1a8006b9aaf4cb21a92.png)
- Перезапустите Obsidian
- **Запустите команду** в Obsidian (CMD/Ctrl + P): `Clone an existing remote repo` (Клонировать существующий удаленный репозиторий).
- **Вставьте URL** вашего репозитория в следующем формате: `https://github.com/advaitateam/advayta.org-docs.git`
- **Введите название папки**, которую вы создали на шаге 2 (например, `advayta-org/`).
- **Перезапустите Obsidian**.
- Внесите изменения в свои заметки.
- **Опубликуйте заметки**, запустив команду `Obsidian Git: Create backup` через палитру команд (CMD/Ctrl + P).
