# Ежедневные задачи

## Рекомендации перед началом работы
- всегда открывайте git плагин и забирайте последние изменений (pull) что б иметь обновленный контент
- просматривайте [историю изменений](https://github.com/advaitateam/advayta.org-docs/commits/master2/) контента и описания изменений

## Отправка нового контента на сервер

- Нажмите иконку git source control слева ![[интрукции/media/Pasted image 20251107143524.png]]
- справа в панели пересмотрите список файлов который вы меняли, убедитесь єто именно то что вы хотите отправить на сервер
- введите описание ваших изменений
- нажмите кнопку commit and sync для отправки изменений
