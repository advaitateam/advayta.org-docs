## Настройка доступов
- Создайте учетную запись в [github](https://github.com/) и сообщите ваш никнайм или емайл Индешваре или Видьяранье

### Сгенерируйте персональный токен в github
- Откройте GitHub, нажмите на аватар справа сверху.
- Перейдите в [settings](https://github.com/settings/profile)
- В левом меню откройте [Developer settings](https://github.com/settings/tokens).
- Откройте Personal access tokens → [Tokens (classic)](https://github.com/settings/tokens/new)
- Нажмите Generate new token.

### Заполните
- Note - obsidian notes
- Expiration - No expiration (очень важно)
- В Select scopes выберите: Repo, все галачки для repo должны быть отмечены all
  ![image](https://storage.googleapis.com/media-thumbs.advayta.org/m/4d2460db6adaa74a035adb6b4926d724.png)
- Проскрольте сайт вниз и **Нажмите** Generate token.

> Сразу скопируйте токен — он на будем нужен на следующем шаге


## Скачивание файлов в папку files
- Откройте Obsidian. Создайте vault c названием `advayta`, откройте настройки, включите галачку community plugin ![image](https://media-thumbs.advayta.org/m/1286a970c5145f73a9cd9bdca886219d.png)
- Нажмите browser, найдите установите плагин git
  ![image](https://media-thumbs.advayta.org/m/ffd4ef8d23d0a1a8006b9aaf4cb21a92.png)
- Перезапустите Obsidian
- **Запустите команду** в Obsidian (CMD/Ctrl + P): `Clone an existing remote repo` (Клонировать существующий удаленный репозиторий).
  ![image](https://storage.googleapis.com/media-thumbs.advayta.org/m/602a6158884ab2e18f453e5161f0bc62.png)
- **Вставьте URL** вашего репозитория в следующем формате: 
```
  https://<PERSONAL_ACCESS_TOKEN>@github.com/advaitateam/advayta.org-docs.git
```
- Замените `<PERSONAL_ACCESS_TOKEN>` на ваш токен который вы создали ранее. Те у вас должна быть вот такая сторочка 
```
  https://ghp_XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX@github.com/advaitateam/advayta.org-docs.git
```
- **Введите название папки**, где будут находиться файлы например, `files`.
- **Создайте новый vault** нажав на надпись `advayta` слева внизу. В меню выберете настройка хранилищ (manage vault)
- Выберете раздел открыть папку как хранилище, нажмите **открыть**
- Найдите и откройте папку `files` созданую ранее
- Obsidian создает новое окно, на вопрос можно ли доверять автору и включить плагины выберете **доверять**
