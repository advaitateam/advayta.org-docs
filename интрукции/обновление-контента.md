## Отправка нового контента на сервер

- Нажмите иконку git source control слева ![image](https://storage.googleapis.com/media-thumbs.advayta.org/m/de091932610bb4b56665df1d50dce3ae.png)
- справа в панели пересмотрите список файлов который вы меняли, убедитесь єто именно то что вы хотите отправить на сервер
- введите описание ваших изменений
- нажмите кнопку commit and sync для отправки изменений ![image](https://storage.googleapis.com/media-thumbs.advayta.org/m/1d5160b129e2548423360be1147049f2.png)
