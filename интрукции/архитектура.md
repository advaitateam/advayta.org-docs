1. Сайт содержит только текстовую информацию, связь между статьями строится по принципу [зеттелкастена](https://www.goodnotes.com/blog/zettelkasten-method). Все храниться в md файлах в [открытом гитхаб](https://github.com/advaitateam/advayta.org-docs) репозитарии
2. Каждая статья должна контцентрироваться на одной конкретной проблеме / термине
3. wiki ccылки помогают связать статьи в цикл статей, что б єто было удобно читать
4. Картинки загружаются автоматически на s3 хранилище через плагин в obsidian (там очень круто что community может писать свои плагины)
5. md контент рендериться через vitepress в html и загружается на cloudflare s3
6. md файлы индексуирется через algolia search который формирует js для удобного и супер быстрого поиска без бекенда
