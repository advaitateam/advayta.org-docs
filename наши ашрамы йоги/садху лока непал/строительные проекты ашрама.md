## Здание келий для монахов-отшельников
Первым строящимся зданием стало небольшое здание с четырьмя кельями.
Оно предназначено для проживания и практики монахов-отшельников.
Планируется завершить строительство осенью 2025 года, чтобы монахи могли полноценно разместиться на постоянной основе.

![Ретритный комплекс для учеников-мирян](https://www.advayta.org/upload/medialibrary/f4f/e7f9v0321d0z694986m5aj67l3pvz9r9.png "Ретритный комплекс для учеников-мирян")

  

## **Ретритный комплекс для учеников-мирян**

Следующим этапом станет возведение ретритного комплекса.
В нём предусмотрено 8–10 комнат для проживания, алтарно-ретритный зал, кухня-столовая и хозяйственные помещения.

Этот комплекс станет местом для длительных ретритов учеников, желающих углубить свою практику в уединённых условиях.

Ожидается, что это будет одно из самых больших зданий на территории ашрама.

![Главное здание Ашрама](https://www.advayta.org/upload/medialibrary/8bc/1wtnx6y30uheyvx8t53sbcq5272xh4m6.png "Главное здание Ашрама")

  

## **Главное здание Ашрама**

Завершающим крупным этапом будет строительство главного здания. В нём разместятся:

- основной алтарный зал
- кухня-столовая для монахов
- монастырская лавка
- приёмная
- комнаты для проживания монахов
